# История изменений

Все значимые изменения в EWSP Core Library будут задокументированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
а проект придерживается [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-08

### Добавлено
- **Blockchain Packet Chaining**: пакеты, связанные через SHA-256, для защиты от повторных атак без временных меток
- **XChaCha20 Encryption**: потоковый шифр с расширенным nonce (24 байта) и разделением ключей
- **HKDF Key Derivation**: деривация ключей по RFC 5869 с доменным разделением
- **HMAC-SHA256 Authentication**: аутентификация сообщений с константно-временной верификацией
- **Кроссплатформенный RNG**: платформо-специфичный безопасный генератор случайных чисел:
  - Windows: CryptGenRandom
  - Linux/macOS: /dev/urandom
  - ESP32/ESP8266: аппаратный RNG esp_random()
- **Protocol v1.0**: полный формат пакетов с шифрованием, подписью и управлением цепочкой
- **JSON Serialization**: встроенный JSON парсер/строитель для обработки пакетов
- **Command System**: расширяемая диспетчеризация команд с типобезопасной сериализацией
- **Обработка ошибок**: исчерпывающие коды ошибок с информативными сообщениями

### Безопасность
- **CRYPTO-03**: разделение ключей через HKDF (ключ шифрования ≠ ключ аутентификации)
- **CRYPTO-04**: константно-временное сравнение для всей верификации MAC/хэшей
- **Безопасная память**: `ewsp_secure_zero()` для очистки конфиденциальных данных
- **Без небезопасных fallback**: ошибка компиляции вместо fallback на rand()

### API
- Все публичные символы используют префикс `ewsp_`
- Полная документация Doxygen для публичного API
- Чёткое разделение: `ewsp_types.h`, `ewsp_errors.h`, `ewsp_crypto.h`, `ewsp_packet.h`

## [1.0.0] - 2025-06-15

### Добавлено
- Первоначальный выпуск
- Реализация хэш-функции SHA-256
- Аутентификация HMAC-SHA256
- Потоковый шифр ChaCha20
- Базовая сериализация пакетов

## Планы развития

### [2.1.0] - Запланировано
- **XChaCha20-Poly1305 AEAD**: аутентифицированное шифрование со связанными данными
- **Session Management**: обмен ключами X25519, прямая секретность, привязка сессий
- **Python Bindings**: ctypes обёртка для Python клиентов
- **Kotlin/JNI Bindings**: нативная поддержка Android

### [3.0.0] - В будущем
- Подписи Ed25519 для обновлений прошивки
- WASM сборка для браузерных клиентов
- Формальный аудит безопасности
